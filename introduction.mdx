# Introduction

Kitsu is a web-based collaboration platform that helps animation, VFX, and video game studios manage their creative productions and deliveries. Built by CGWire, Kitsu improves communication between all stakeholders, leading to better results and faster project completion.

## Who uses Kitsu?

More than 300 studios around the world use Kitsu to manage their projects. The platform is designed for:

- **Animation studios** producing films, series, and short-form content
- **VFX studios** working on visual effects for film and television
- **Video game studios** managing asset creation and production pipelines
- **Production teams** of all sizes, from small independent studios to large-scale operations

## Main features

Kitsu provides a comprehensive suite of tools for production management:

### Production tracking
- Create and organize productions with customizable workflows
- Track assets, shots, and sequences throughout the production pipeline
- Manage task assignments and monitor progress in real-time

### Review and feedback
- Built-in annotation and drawing tools for precise feedback
- Version control for assets and shots
- Collaborative review sessions with stakeholders

### Team collaboration
- Real-time updates via WebSocket connections
- Comment threads and mentions for team communication
- Calendar view for scheduling and deadlines

### Reporting and analytics
- Production dashboards with charts and metrics
- Time tracking and estimation tools
- Export capabilities to Excel and other formats

### Customization
- Flexible metadata and custom fields
- Multi-language support (internationalization ready)
- Integration capabilities via API

## Technology stack

Kitsu is built with modern web technologies:

- **Frontend**: Vue.js 3 with Vuex for state management
- **Styling**: Bulma CSS framework
- **Build tool**: Vite for fast development and optimized builds
- **Real-time**: Socket.io for live updates
- **Visualization**: Chart.js for production metrics
