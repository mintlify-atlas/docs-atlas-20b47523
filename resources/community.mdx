## Get Support

### Community Support

The Kitsu community is friendly and helpful. Get support through:

<Steps>
  <Step title="Join Discord">
    [Join the Discord server](https://discord.com/invite/VbCxtKN) where community members and the CGWire team provide support.
  </Step>
  <Step title="Search Discussions">
    Check if your question has already been answered in previous conversations.
  </Step>
  <Step title="Ask Your Question">
    Post your question with details about your setup, issue, and what you've tried.
  </Step>
  <Step title="Help Others">
    Share your knowledge by helping other community members when you can.
  </Step>
</Steps>

### Report Issues

Found a bug? Help improve Kitsu by reporting it:

<Card title="GitHub Issues" icon="bug" href="https://github.com/cgwire/kitsu/issues">
  Report bugs and track issues on our GitHub repository. Before creating a new issue, please search existing issues to avoid duplicates.
</Card>

<Tip>
  When reporting a bug, include:
  - Clear description of the issue
  - Steps to reproduce
  - Expected vs actual behavior
  - Environment details (OS, browser, Kitsu version)
  - Screenshots or error messages if applicable
</Tip>

### Code Contributions

Contribute code to Kitsu:

<Steps>
  <Step title="Find a Task">
    Browse [GitHub Issues](https://github.com/cgwire/kitsu/issues) or [feature requests](https://cgwire.canny.io/) for something to work on.
  </Step>
  <Step title="Follow C4 Contract">
    All contributions must respect the [C4 contract](https://rfc.zeromq.org/spec:42/C4) which defines the contribution process.
  </Step>
  <Step title="Set Up Development Environment">
    Follow the [Development Environment](/development) guide to set up your local environment.
  </Step>
  <Step title="Submit Pull Request">
    Create a pull request with your changes. Include a clear description of what you've done and why.
  </Step>
</Steps>

<Info>
  Kitsu is built with:
  - JavaScript (ES6+)
  - Vue.js 3 framework
  - Vuex for state management
  - Vite for build tooling
  
  Familiarity with these technologies will help you contribute effectively.
</Info>

## Code of Conduct

Our community is built on mutual respect and collaboration. When participating in the Kitsu community:

- Be respectful and inclusive
- Help others when you can
- Give constructive feedback
- Assume good intentions
- Follow the [C4 contract](https://rfc.zeromq.org/spec:42/C4) for code contributions

We want everyone to feel welcome and supported in the Kitsu community.
