## Vision

Kitsu aims to be the best open-source collaboration platform for animation, VFX, and video game studios worldwide. Our roadmap is guided by community feedback, industry needs, and our vision for streamlined creative production.

<Info>
  This roadmap represents our current plans and priorities. Timelines and features may change based on community feedback and development priorities.
</Info>

## How We Prioritize

Our development priorities are influenced by:

1. **Community Feedback** - Feature requests and votes on [Canny](https://cgwire.canny.io/)
2. **User Needs** - Direct feedback from our 300+ studio users
3. **Industry Trends** - Evolving needs in animation, VFX, and game production
4. **Technical Debt** - Maintaining code quality and performance
5. **Security** - Keeping the platform secure and up-to-date

## Technology Updates

### Current Technology Stack

- Vue.js 3.5.13
- Vite 7.3.1
- Node.js 20.19+
- Modern JavaScript (ES6+)

### Planned Updates

<Steps>
  <Step title="Framework Updates">
    Keep Vue.js and core dependencies updated to latest stable versions for security and performance.
  </Step>
  
  <Step title="Build System">
    Continuous optimization of Vite configuration for faster builds and better development experience.
  </Step>
  
  <Step title="Testing Infrastructure">
    Expand test coverage with Vitest to ensure stability and prevent regressions.
  </Step>
  
  <Step title="TypeScript Migration">
    Gradual migration to TypeScript for better type safety and developer experience.
  </Step>
</Steps>

## Release Strategy

Our typical release cycle:

<Steps>
  <Step title="Feature Development">
    Develop new features based on roadmap priorities and community feedback.
  </Step>
  
  <Step title="Testing Phase">
    Thorough testing including unit tests, integration tests, and user acceptance testing.
  </Step>
  
  <Step title="Release Candidate">
    RC version available for community testing and feedback.
  </Step>
  
  <Step title="Stable Release">
    Final release after incorporating feedback and fixing any discovered issues.
  </Step>
</Steps>

## Long-term Vision

### Studio of the Future

We envision Kitsu as the central hub for creative production:

- **Unified Platform**: Single source of truth for all production data
- **Smart Automation**: AI-assisted task assignment and scheduling
- **Seamless Integration**: Works perfectly with all your favorite tools
- **Scalable**: From small studios to large enterprises
- **Accessible**: Easy to use for both technical and non-technical users

### Open Source Commitment

Kitsu will always remain open source under AGPL-3.0:

- Transparent development process
- Community-driven priorities
- No vendor lock-in
- Freedom to customize and extend
- Collaborative improvement
