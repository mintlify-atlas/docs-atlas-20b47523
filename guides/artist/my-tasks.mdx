## Overview

The My Tasks page is your central workspace in Kitsu. Here you can view all tasks assigned to you, track your progress, and manage your daily workload across different productions.

## Accessing Your Tasks

Navigate to **My Tasks** from the main navigation menu. You'll see several tabs to organize your work:

- **Tasks** - Your active assignments
- **Board** - Kanban-style view of tasks by status
- **Calendar** - Visual timeline of tasks with due dates
- **Pending** - Tasks awaiting feedback or review
- **Validated** - Completed and approved tasks
- **Timesheets** - Time tracking for your work
- **Days Off** - Manage your time off

## Understanding the Task List

Each task row displays:

- **Production** - Project avatar/icon
- **Task Type** - The type of work (e.g., Modeling, Animation, Compositing)
- **Entity** - The asset or shot name with thumbnail preview
- **Episode** - For TV show productions
- **Estimation** - Expected hours/days for completion
- **Duration** - Time already spent on the task
- **Start Date** - When work should begin
- **Due Date** - Deadline for completion
- **Status** - Current task status with color coding
- **Last Comment** - Latest feedback or update

<Tip>
Tasks with red duration values indicate you've exceeded the estimated time. This helps you identify tasks that need attention or re-estimation.
</Tip>

## Filtering and Sorting Tasks

### Production Filter

Use the production dropdown to focus on tasks from a specific project, or select "All" to see tasks across all your productions.

### Show Filter

- **All tasks** - Display everything assigned to you
- **Due this week** - Focus on urgent work with upcoming deadlines

### Sort Options

Organize your task list by:

- **Priority** - Show highest priority tasks first (default)
- **Entity name** - Alphabetical by asset/shot name
- **Task status** - Group by current status
- **Start date** - Chronological by start date
- **Due date** - Order by deadline
- **Estimation** - Sort by estimated work time
- **Last comment date** - Show recently updated tasks first

## Search Functionality

The search field allows you to:

- Search by entity name, task type, or status
- Save frequently used searches for quick access
- Use advanced filters to narrow down results

### Saving a Search

1. Enter your search criteria
2. Click the save icon
3. Name your search query
4. Access it later from the saved searches list

## Selecting Tasks

Click on any task row to select it. The task details will appear in the side panel, showing:

- Task information and metadata
- Preview files and comments
- Assignees
- Status history

### Multiple Selection

- **Click** - Select a single task
- **Ctrl/Cmd + Click** - Add tasks to selection
- **Shift + Click** - Select a range of tasks

## Task Status Indicators

Status colors indicate the current state:

- **Gray** - Not started (Todo)
- **Blue** - Work in progress
- **Purple** - Ready for review
- **Orange** - Retake/revision needed
- **Green** - Approved/validated
- **Red** - Blocked or requiring attention

<Note>
Status colors and names are customizable by your production team, so they may vary between projects.
</Note>

## Board View

Switch to the **Board** tab for a Kanban-style interface:

- Tasks are organized in columns by status
- Drag and drop tasks between status columns to update them
- Get a visual overview of your workflow
- Filter by production to focus on specific projects

## Calendar View

The **Calendar** tab provides:

- Monthly view of tasks with start and due dates
- Color-coded by task status
- Click on any day to see tasks scheduled for that date
- Visual identification of scheduling conflicts

## Pending Tasks

The **Pending** tab shows tasks where:

- You've submitted work for review
- Status is set to "Waiting for Approval" or similar feedback states
- Supervisors need to provide feedback

Check this tab regularly for updates on tasks awaiting validation.

## Validated Tasks

The **Validated** tab displays:

- Completed and approved tasks
- End date when task was closed
- Final comments and approvals
- Historical record of your completed work

<Tip>
You can still log time on validated tasks if you need to add hours worked before the task was closed.
</Tip>

## Keyboard Navigation

For faster navigation:

- **Alt + Arrow Up/Down** - Navigate through task list
- **Alt + Arrow Left/Right** - Move between tasks
- Selection follows your keyboard navigation

## Time Tracking Statistics

At the bottom of the task list, you'll see:

- **Total tasks** in current view
- **Time spent** - Sum of all duration values
- **Time estimated** - Sum of all estimations
- Hours or days depending on your organization's settings

This helps you understand your workload and track progress toward estimates.

## Working with Task Details

When you select a task:

1. **View Preview** - See the latest work submitted
2. **Add Comments** - Communicate with your team
3. **Upload Work** - Submit new preview files (see [Uploading Work](/guides/artist/uploading-work))
4. **Check Status** - Review current state and history
5. **Log Time** - Record hours worked (see [Timesheet](/guides/artist/timesheet))
