## Overview

Tracking production progress is essential for hitting deadlines and maintaining quality. Kitsu provides multiple views and tools to monitor status, identify issues early, and make data-driven decisions.

## Production Dashboard

### Accessing Production Overview

Navigate to your production from the main menu to access:

- **Assets**: All production assets organized by type
- **Shots**: Shot breakdown by sequence and episode
- **Task Status**: Visual grid showing completion across task types
- **Statistics**: Production-wide metrics and progress

### Task Status Grid

The main assets and shots pages display a grid where:

- **Rows** = Individual assets/shots
- **Columns** = Task types (Modeling, Rigging, Animation, etc.)
- **Cells** = Status color for each task

**Color Coding**:
- **Gray**: Not started / Todo
- **Blue**: Work in progress
- **Purple**: Waiting for approval / Pending
- **Orange**: Retake needed
- **Green**: Approved / Complete
- **Red**: Blocked / Issues

<Tip>
Get a quick production overview by scanning the color grid. Lots of gray means work hasn't started; lots of orange means issues need addressing.
</Tip>

## Filtering and Views

### Production Filters

Narrow your view with filters:

- **By Asset Type**: Props, Characters, Environments, etc.
- **By Episode**: For episodic productions
- **By Sequence**: Group shots logically
- **By Task Status**: Show only specific statuses
- **By Assignee**: See work for specific artists

### Search Functionality

Quickly find specific items:

- Search by name (asset/shot code)
- Filter by metadata (complexity, priority, etc.)
- Combine filters for precise queries
- Save custom searches for reuse

### Column Visibility

Customize which task columns you see:

1. Click the column visibility menu
2. Toggle task types on/off
3. Focus on your department's tasks
4. Adjust based on production phase

## Progress Statistics

### Entity-Level Stats

At the top of asset/shot pages, see:

- **Total entities**: Count of assets or shots
- **Completion percentage**: Overall progress
- **Task counts** by status
- **Estimated vs. actual** time

### Task-Level Breakdown

For each task type column:

- Click the column header
- View statistics modal
- See status distribution
- Track approval rates

### Production Metrics

Access detailed analytics:

- **Quota page**: Daily/weekly completion rates
- **Time spent**: Aggregated hours by task type
- **Estimation accuracy**: Planned vs. actual
- **Bottleneck identification**: Tasks taking longest

<Note>
Production metrics help you spot trends, like consistently underestimated task types or departments running behind.
</Note>

## Tracking by Task Type

### Department Progress

Monitor progress for your specific area:

1. Filter to your task type (e.g., Animation)
2. Review status distribution
3. Identify tasks needing attention
4. Check for overdue items

### Task Status Breakdown

For your task type, track:

- **Not started**: How much work is queued
- **In progress**: Currently active tasks
- **Pending review**: Your review queue
- **Retakes**: Issues to follow up on
- **Approved**: Completed work

### Common Patterns to Watch

**Red Flags**:
- Many tasks stuck in retake status
- Low approval rate (lots of iterations)
- Tasks not progressing over time
- Overdue tasks piling up

**Green Lights**:
- Steady progression through statuses
- High first-pass approval rate
- Tasks completing on or ahead of schedule
- Balanced workload across artists

## Schedule Management

### Viewing Schedules

Access the **Schedule** page to see:

- Gantt-style timeline view
- Tasks plotted by start and due dates
- Dependencies between tasks
- Milestones and deadlines

### Schedule Types

Kitsu supports different schedule views:

1. **Main Schedule**: Production-wide timeline
2. **Asset Type Schedules**: Modeling, rigging, etc.
3. **Sequence Schedules**: Shot progression
4. **Department Schedules**: Team-specific views

### Schedule Items

Each schedule item shows:

- Task type and entity
- Planned duration (from estimation)
- Start and end dates
- Current status
- Assignee(s)

### Identifying Schedule Issues

Look for:

- **Red bars**: Overdue tasks
- **Compressed timelines**: Insufficient time for quality work
- **Gaps**: Idle time or missing assignments
- **Conflicts**: Overlapping critical tasks

<Warning>
Tasks showing as overdue need immediate attention. Review with the assigned artist and adjust deadlines or resources as needed.
</Warning>

## Quota Tracking

### What Is Quota?

Quota represents expected task completion rates:

- Tasks per day/week/month
- Compared to actual completions
- Shows team velocity
- Helps forecast delivery dates

### Accessing Quota Page

From your production menu:

1. Select **Quota**
2. Choose date range (daily, weekly, monthly)
3. View completion charts
4. Compare to targets

### Quota Metrics

The quota page shows:

- **Target**: Expected completions
- **Actual**: Tasks completed
- **Variance**: Ahead or behind schedule
- **Trends**: Performance over time
- **By task type**: Department breakdowns

### Using Quota Data

Quota helps you:

- Forecast delivery dates based on current velocity
- Identify if team needs more resources
- Spot productivity trends (improving or declining)
- Set realistic expectations with production

<Tip>
If quota shows consistent under-delivery, investigate root causes: Are estimations too optimistic? Is the team size adequate? Are there blockers?
</Tip>

## Time Tracking Analysis

### Team Timesheets

Access the **Timesheets** page to review:

- Time logged by all team members
- Daily, weekly, monthly views
- Breakdown by production and task type
- Compare estimated vs. actual hours

### Identifying Estimation Issues

Look for patterns:

- **Consistently over**: Tasks taking much longer than estimated
- **Consistently under**: Estimations are too conservative
- **High variance**: Inconsistent scoping

### Capacity Planning

Use timesheet data to:

- Calculate available team hours
- Estimate completion dates
- Identify overallocation
- Plan future assignments

**Example**:
```
Team size: 5 artists
Available hours/week: 40 hours × 5 = 200 hours
Remaining task estimations: 800 hours
Estimated completion: 4 weeks
```

## Identifying Bottlenecks

### Common Bottleneck Signs

1. **Tasks stuck in same status** for extended periods
2. **Reviewer backlog**: Many tasks waiting for approval
3. **Dependency chains**: One delayed task blocking many others
4. **Resource constraints**: Same person assigned to too many tasks
5. **Technical issues**: Repeated retakes for same reason

### Resolving Bottlenecks

<AccordionGroup>
  <Accordion title="Review backlog">
    - Dedicate time for reviews daily
    - Delegate review authority when possible
    - Use batch review workflows
    - Communicate expected review turnaround
  </Accordion>
  
  <Accordion title="Dependency issues">
    - Identify critical path tasks
    - Prioritize blocking work
    - Consider parallel workflows
    - Communicate dependencies clearly
  </Accordion>
  
  <Accordion title="Resource overload">
    - Redistribute tasks
    - Bring in additional help
    - Reduce scope if possible
    - Adjust deadlines
  </Accordion>
  
  <Accordion title="Technical problems">
    - Identify recurring issues
    - Provide training or documentation
    - Improve tools or pipelines
    - Clarify standards and expectations
  </Accordion>
</AccordionGroup>
